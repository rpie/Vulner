# Vulner
Vulner is a website vulnerability scanner and auto exploiter
You can use this tool to check the security of your or others websites.
Please do not use this on others websites without permission.

# Install


### Linux
```unix
git clone https://github.com/rpie/Vulner/
cd Vulner/
python3 install.py linux
python3 main.py
```


### Windows

```powershell
curl "https://github.com/rpie/Vulner/archive/refs/heads/main.zip" -O Vulner
cd Vulner/
python install.py windows
python main.py 
```


## Mac
```bash
git clone https://github.com/rpie/Vulner/
cd Vulner/
python3 install.py mac
python3 main.py
```
